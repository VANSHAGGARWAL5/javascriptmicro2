This project is a simple MEN application that demonstrates how to connect 
Node.js with MongoDB Atlas using Mongoose, implement a schema with validation, 
and perform a CREATE operation using POSTMAN.

## Features
- ES Module syntax
- MEN folder structure (config, models, controllers, routes)
- MongoDB Atlas connection with environment variables
- Schema with validation rules
- Create (POST) API endpoint
- Organized and clean code

## How to Run
npm install
npm run dev

## API Endpoint
POST /api/items  
Sends JSON body to create a new item in MongoDB.
